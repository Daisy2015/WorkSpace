# 多智能体对话区完整演示

## 概述

本系统实现了完整的 **Agentic 模式** 对话流程，展示了从用户提问到 Leader 编排、岗位拆解、场景执行、阶段汇报、最终结论的完整链路。

## 完整演示：X-1井产量下降分析

### 🎯 演示场景

**业务背景**: 油田生产分析
**用户需求**: 分析 X-1井近7天产量下降原因，并给出未来3天稳产优化建议
**涉及角色**: 生产分析岗智能体

***

## 执行流程（8帧）

### 第0帧：用户提问

**组件**: `UserMessageCard`

```
用户输入：
"请分析 X-1井近7天产量下降原因，并给出未来3天稳产优化建议。"

左侧引用资源（6个）：
├── 井: X-1井
├── 数据集: 生产日报
├── 数据集: 历史措施记录
├── 井组: 邻井对比井组
├── 图件: 实时压力曲线
└── 图件: 含水率趋势图
```

**视觉特点**:

- 白色卡片，Indigo-600 强调色
- 资源标签可点击，联动左侧资源树
- Hover 显示编辑/重试/复制操作

## 技术特性

### 1. 完整 React 循环

每个 LoopCard 都严格遵循：

```
Thought（思考）→ Action（执行）→ Observation（观察）→ Plan（下一步）
```

### 2. 工具链可视化

- 通用智能体可展开显示内部工具调用
- 缩进式层级展示
- 工具图标标识

### 3. 阶段答案机制

- 在关键节点插入 StageResultCard
- 给用户即时反馈
- 支持基于阶段结果继续分析

### 4. 状态感知

- 执行中/已完成/异常/等待
- 实时进度条
- 流式更新支持
